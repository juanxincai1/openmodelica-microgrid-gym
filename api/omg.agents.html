class openmodelica_microgrid_gym.agents.Agent(obs_varnames=None, history=None, env=None)[source]¶

Bases: object

Abstract base class for all Agents. The agent can act on the environment and observe its result. This class is aims to wrap the whole learning process into a class to simplify the implementation.

This class is strongly inspired by the tensorforce library.

Parameters

  • obs_varnames (Optional[List[str]]) – list of variable names that match the values of the observations passed in the act function. Will be automatically set by the Runner class

  • history (Optional[EmptyHistory]) – used to store agent internal data for monitoring

  • env (Optional[ModelicaEnv]) – reference to the environment (only needed when used in internal act function)

class openmodelica_microgrid_gym.agents.SafeOptAgent(mutable_params, abort_reward, min_performance, kernel, gp_params, ctrls, obs_template, obs_varnames=None, **kwargs)[source]¶

Bases: openmodelica_microgrid_gym.agents.staticctrl.StaticControlAgent, openmodelica_microgrid_gym.agents.episodic.EpisodicLearnerAgent

Agent to execute safeopt algorithm (https://arxiv.org/abs/1509.01066) to control the environment by using auxiliary controllers and Gaussian process to adopt the controller parameters (mutable_params) to safely increase the performance.

Parameters

  • mutable_params (Union[dict, list]) – safe inital controller parameters to adopt

  • abort_reward (int) – factor to multiply with the initial reward to give back an abort_reward-times higher negative reward in case of limit exceeded

  • kernel (Kern) – kernel for the Gaussian process unsing GPy

  • gp_params (Dict[str, Any]) – kernel parameters like bounds and lengthscale

  • ctrls (List[Controller]) – Controllers that are feed with the observations and exert actions on the environment

  • obs_template (Mapping[str, List[Union[List[str], ndarray]]]) –

    Template describing how the observation array should be transformed and passed to the internal controllers. The key must match the name field of an internal controller. The values are a list of:

    • list of strings

      • matching variable names of the state

      • must match self.obs_varnames

      • will be substituted by the values on runtime

      • will be passed as an np.array of floats to the controller

    • np.ndarray of floats (to be passed statically to the controller)

    • a mixture of static and dynamic values in one parameter is not supported for performance reasons.

    The values will be passed as parameters to the controllers step function.

  • obs_varnames (Optional[List[str]]) – list of variable names that match the values of the observations passed in the act function. Will be automatically set by the Runner class :param min_performance: Minial allowed performance to define parameters as safe
