omg.util.randproc¶

class openmodelica_microgrid_gym.util.randproc.RandProcess(process_cls, proc_kwargs=None, bounds=None, initial=0)[source]¶

Bases: object

wrapper around stochastic processes to allow easier integration

Parameters

  • process_cls (Type[BaseProcess]) – class of the stochastic process

  • proc_kwargs – arguments passed to the class on initialization

  • bounds – boundaries of admissible values

  • initial – starting value of the process

  • gain – scale of the process output (of not set, results are between the clipped bounds)

property reserve¶

This variable is used to prepare for external loadsteps or other abrupt changes in the process’ variables.
