class openmodelica_microgrid_gym.aux_ctl.inverter_controllers.MultiPhaseDQCurrentController(IPIParams, pllPIParams, i_limit, Pdroop_param, Qdroop_param, ts_sim, ts_ctrl=None, lower_droop_voltage_threshold=150.0, *args, **kwargs)[source]¶

Bases: openmodelica_microgrid_gym.aux_ctl.inverter_controllers.CurrentCtl

Implements a discrete 3-phase current sourcing inverter, using a PLL to keep track of the external phase angle

Controls the currents dq0 axis, aligned to the external voltage vector, d-axis is aligned with the A phase. Rotating frame aligned with A axis at # t = 0, that is, at t = 0, the d-axis is aligned with the a-axis.

DOES NOT wait for PLL lock before activating

Parameters

  • IPIParams (PI_params) – PI_parameter for the current control loop

  • pllPIParams (PLLParams) – PI parameters for the PLL controller

  • i_limit (float) – Current limit

  • Pdroop_param (InverseDroopParams) – Droop parameters for P-droop

  • Qdroop_param (InverseDroopParams) – Droop parameters for Q-droop

  • lower_droop_voltage_threshold (float) – Grid voltage threshold from where the controller starts to react on the voltage and frequency in the grid

  • ts_sim (float) – positive float. absolute time resolution of the env

  • ts_ctrl (Optional[float]) – positive float. absolute sampling time for the controller

  • history – Dataframe to store internal data

class openmodelica_microgrid_gym.aux_ctl.inverter_controllers.MultiPhaseDQCurrentSourcingController(IPIParams, ts_sim, ts_ctrl=None, f_nom=50.0, *args, **kwargs)[source]¶

Bases: openmodelica_microgrid_gym.aux_ctl.inverter_controllers.Controller

Implements a discrete multiphase PI current controller to supply an amount of current to a load / the grid which can be chosen via state”extension” using idq0SP at a frequency of f_nom. Has its own internal oscillator to keep track of the internal angle.

Controls each phase individually in the dq0 axis.

Parameters

  • IPIParams (PI_params) – PI_parameter for the current control loop

  • ts_sim (float) – positive float. absolute time resolution of the env

  • ts_ctrl (Optional[float]) – positive float. absolute sampling time for the controller

  • f_nom (float) – nominal frequency for current sourcing ctrl
